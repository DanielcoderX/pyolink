Pyolink is Front page generator from GitHub repositories

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Pyolink requirements.

```bash
pip install -r requirements.txt
```

## Usage

```bash
python pyolink.py --username `GitHub username` --theme (optional(currently first theme is available))
```
